# GSD Skill Preferences

## UI/UX Design

When any task involves **UI structure, visual design decisions, interaction patterns, or user experience quality control**, prefer the `ui-ux-pro-max` skill over `frontend-design`.

This includes:
- Designing new pages (landing, dashboard, detail views, editors)
- Creating or refactoring UI components (buttons, modals, forms, tables, cards)
- Choosing color schemes, typography, spacing, or layout systems
- Reviewing UI code for UX, accessibility, or visual consistency
- Implementing navigation, animations, or responsive behavior
- Making product-level design decisions (style, hierarchy, brand expression)
