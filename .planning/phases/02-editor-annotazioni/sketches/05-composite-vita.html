Composite Vita component

Coppia nascita + morte (opt) ottimizzata per spazio iPad reale.

React mockup vero

Questo sketch è renderizzato dal dev server

A differenza degli sketch 1-4 e 6 (HTML+CSS replica HeroUI), il composite Vita usa HeroUI v3 + Tailwind v4 reali via mockup React in src/ui/mockups/CompositeVitaMockup.tsx. Questo permette di vedere la densità e i touch target esatti che renderà la app prod, anche su iPad reale.

Come aprirlo
  1. 1
    Lancia il dev server da ~/dev/board: 
    npm run dev
    Vite parte su localhost:5173 senza aprire il browser.
  2. 2
    Apri nel browser: 
    http://localhost:5173/dev/mockup-composite-vita
    Route bypassa auth, accesso diretto al review.
  3. 3
    Per testare su iPad reale (stessa LAN): 
    npm run dev -- --host
# poi su iPad: http://<tuo-ip>:5173/dev/mockup-composite-vita
Da rimuovere prima di prod: la route /dev/mockup-* e il file src/ui/mockups/ sono dev-only artifacts. Cleanup in plan 02-03.
